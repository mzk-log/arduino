第7回：終わらない授業「loop」のパワー

1. テーマ

授業（loop）がくり返されるたびに変数の中身を増やし、増え続ける数字を画面に出し続ける仕組みを学びます。

2. 今回覚えること

💡 はじめに読むミニ知識：

ここに出てくる言葉は、このあとの説明やこれからのレッスンで何度も登場します。だから、今は「へぇ、そんな名前の言葉があるんだな」と眺めるだけで、完全に覚えられなくても全然大丈夫です！安心して進めてくださいね。

  • 増加（ぞうか）： 変数の値を大きくしていくこと。
  • count = count + 1;： いまの値に1を足して、同じ箱へ入れ直す書き方。
  • 状態（じょうたい）： いま変数に入っている値のこと。くり返しのあいだも覚えています。

3. 事前解説

第2回で学んだ通り、loop（授業）は終わるとまた最初から始まります。今回はその力を使って、回るたびに数字が増えるカウンターを作ります。

count = count + 1; は、「いまの count に 1 を足して、もう一度 count に入れる」という意味です。ノートの点数を1点ずつ増やしていくイメージです。

変数を setup（朝の会）の外で作っておくと、授業がくり返されても箱そのものは消えず、中身がどんどん更新されていきます。

4. コード（プログラム）

初めての方・途中から始める方へ：インデントとカッコのルールを確認

下のアルファベットと記号を、パソコンの画面にそのまま打ち込んでみましょう。

int count = 0;

void setup() {
  Serial.begin(9600);
  Serial.println("カウンター開始");
}

void loop() {
  count = count + 1;
  Serial.println(count);
  delay(1000);
}

5. 動かし方（手順）

入力したプログラムを、実際に動かしてみましょう！

手順 1 / 準備
Arduinoをパソコンにつなぐ

USBケーブルを使って、Arduinoとパソコンをしっかりとつなぎます。

手順 2 / チェック
チェックボタン（検証）を押す

画面の左上にある「レ点（✓）のボタン」をクリックします。これは、プログラムに文字の間違いがないかをパソコンが自動で調べるボタンです。画面の下の方に「コンパイル完了」と出れば、間違いがないという証拠です！

手順 3 / 書き込み
矢印ボタン（書き込み）を押す

間違いがないことを確認したら、隣にある「右矢印（➔）のボタン」をクリックします。プログラムがArduinoへ送り込まれます。画面の下の方に「書き込みが完了しました」と出るまで少し待ちます。

手順 4 / 確認
シリアルモニタを開く

画面の右上にある「虫眼鏡（🔍）のボタン」をクリックして、シリアルモニタという新しい画面を開きます。

🔍 確認しよう：

「カウンター開始」のあと、1、2、3… とおよそ1秒おきに増えていったら大成功です！

6. コード解説

  • int count = 0;
    setup の外で変数を作ります。こうすると朝の会でも授業でも使えます。
  • void setup()
    通信準備と開始メッセージを1回だけ行います。
  • count = count + 1;
    いまの値に1を足して代入します。
  • Serial.println(count);
    更新された数字を表示します。
  • delay(1000);
    1秒待ってから次のくり返しへ進みます。
  • void loop()
    この中が終わると、また最初から実行されます。
  • ; と { }
    区切りと範囲です。

7. 練習問題

count = count + 1; を count = count + 2; に変えて、2、4、6… と増えるか確かめてみましょう。

8. うまく動かないとき（エラー）のアクション

もし画面にエラー（赤い文字など）が出て動かない場合は、あわてずに以下の 3つのポイント を手元と見比べて、直してみましょう。

1. 変数は setup の外にありますか？
loop の中だけで int count = 0; を書くと、毎回0に戻ってしまいます。
2. セミコロンを忘れていませんか？
count = count + 1 の最後に ; があるか確認してください。
3. delay は半角ですか？
delay(1000); のつづりと数字を確認してください。
