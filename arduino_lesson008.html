第8回：キーボードの入力を受け取ろう

1. テーマ

パソコンのキーボードで打った文字データをArduinoが受け取り、そのままシリアルモニタへ返す仕組みを学びます。

2. 今回覚えること

💡 はじめに読むミニ知識：

ここに出てくる言葉は、このあとの説明やこれからのレッスンで何度も登場します。だから、今は「へぇ、そんな名前の言葉があるんだな」と眺めるだけで、完全に覚えられなくても全然大丈夫です！安心して進めてくださいね。

  • 入力（にゅうりょく）： キーボードなどからコンピューターへデータを送ること。
  • Serial.available()： 受け取れる文字データが何文字あるかを調べる関数。
  • Serial.read()： 送られてきた文字を1文字読み取る関数。
  • char（チャー）： 文字1つを入れるための型。

3. 事前解説

これまでの回は、Arduinoからパソコンへ文字を送る練習でした。今回は逆に、キーボードからArduinoへ文字を受け取る練習です。

シリアルモニタ下部の入力らんに文字を打ち、送信すると、その文字データがArduinoへ届きます。Serial.read() で1文字ずつ受け取れます。

  • 先に Serial.available() で「届いている文字があるか」を調べます。
  • あるときだけ Serial.read() で読みます。

ここでの if は「もし届いている文字があれば」という最低限の使い方です。くわしい条件分岐は第9回で学びます（今回は先取りです）。

4. コード（プログラム）

初めての方・途中から始める方へ：インデントとカッコのルールを確認

下のアルファベットと記号を、パソコンの画面にそのまま打ち込んでみましょう。

void setup() {
  Serial.begin(9600);
  Serial.println("好きな文字を送ってね");
}

void loop() {
  if (Serial.available() > 0) {
    char c = Serial.read();
    Serial.print("受け取った文字: ");
    Serial.println(c);
  }
}

5. 動かし方（手順）

入力したプログラムを、実際に動かしてみましょう！

手順 1 / 準備
Arduinoをパソコンにつなぐ

USBケーブルを使って、Arduinoとパソコンをしっかりとつなぎます。

手順 2 / チェック
チェックボタン（検証）を押す

画面の左上にある「レ点（✓）のボタン」をクリックします。これは、プログラムに文字の間違いがないかをパソコンが自動で調べるボタンです。画面の下の方に「コンパイル完了」と出れば、間違いがないという証拠です！

手順 3 / 書き込み
矢印ボタン（書き込み）を押す

間違いがないことを確認したら、隣にある「右矢印（➔）のボタン」をクリックします。プログラムがArduinoへ送り込まれます。画面の下の方に「書き込みが完了しました」と出るまで少し待ちます。

手順 4 / 確認
シリアルモニタを開く

画面の右上にある「虫眼鏡（🔍）のボタン」をクリックして、シリアルモニタという新しい画面を開きます。

🔍 確認しよう：

シリアルモニタ下の入力らんに文字（例：A）を打ち、送信ボタン（またはEnter）を押すと、「受け取った文字: A」と返ってきたら大成功です！（改行の記号も一緒に届くことがあります）

6. コード解説

  • Serial.println("好きな文字を送ってね");
    入力を促すメッセージです。
  • if (Serial.available() > 0) { ... }
    受け取れる文字が1つ以上あるときだけ、中の処理を実行します。
  • Serial.available()
    まだ読んでいない文字が何個あるかを返します。
  • char c = Serial.read();
    届いた文字を1つ読み取り、変数 c に入れます。
  • Serial.print(...);
    改行せずに文字を表示します。
  • Serial.println(c);
    受け取った文字を表示して改行します。
  • >
    「より大きい」を表す記号です。
  • void setup() / void loop() と ; { }
    朝の会・授業、区切り、範囲です。

7. 練習問題

表示を 「受け取ったよ: 」 に変えて、同じように文字が返ってくるか確かめてみましょう。

8. うまく動かないとき（エラー）のアクション

もし画面にエラー（赤い文字など）が出て動かない場合は、あわてずに以下の 3つのポイント を手元と見比べて、直してみましょう。

1. シリアルモニタの送信設定を確認しましたか？
入力らんの近くで「改行なし」や「改行」などを選べます。まずは好きな文字を送ってみましょう。
2. なみカッコはそろっていますか？
if の { } と loop の { } を確認してください。
3. セミコロンを忘れていませんか？
char c = Serial.read() の最後に ; があるか確認してください。
