Arduino Lesson

小・中学生のためのArduinoプログラミング講座

パソコンとArduinoで、プログラムの基本から電子工作まで段階的に学ぼう。
自分に合ったコースから、一つずつ「できた！」を増やしていきます。

初めての方は「はじめる前の準備」から進みましょう。

はじめる前の準備

Arduino IDEのインストール、基板とUSBケーブルの準備、ボード・ポートの設定、最初の書き込みを確認します（推奨基板：Arduino Nano）。

学習ガイド

レッスン中に分からなくなったときや、もっと詳しく知りたいときに使います。

コードを書く基本ルール インデントやカッコの役割を確認します。 データ型を知ろう intやcharの違いを整理します。 リファレンスで調べよう 関数や型を自分で探す方法を学びます。

基礎編

外部部品を使わず、シリアルモニタを中心に学び、後半でオンボードLEDとプロッタも扱います。全14回です。

第1回 パソコンの画面にメッセージを出してみよう！ はじめる → 第2回 プログラムの２つの行事（朝の会 と 授業） はじめる → 第3回 時間をあやつる命令を使おう（delay） はじめる → 第4回 データをあずかる「魔法の箱」（変数） はじめる → 第5回 箱の中身を入れ替えよう（代入） はじめる → 第6回 コンピューターに計算させよう（演算子） はじめる → 第7回 終わらない授業「loop」のパワー はじめる → 第8回 キーボードの入力を受け取ろう はじめる → 第9回 もし〜なら（条件分岐 if） はじめる → 第10回 そうでなければ（ if - else ） はじめる → 第11回 決まった回数だけ繰り返そう（ for ） はじめる → 第12回 総合演習「入力で切りかわる『クイズ・ボット』を作ろう」 はじめる → 第13回 Arduino本体のLEDを点滅させよう はじめる → 第14回 LEDの点滅を画面でも見よう（シリアルモニタ／プロッタ） はじめる →
