💡 学習のコツ：

わからなくても大丈夫。まずは動かしてみて、「できた！」を積み重ねていきましょう。保護者と一緒に読むのもおすすめです。

📘 初めての方へ：

コードを書く前に、インデントとカッコの基本ルールを読んでおきましょう。途中の回から始める人も、ここから確認できます。

第1回 パソコンの画面にメッセージを出してみよう！ はじめる → 第2回 プログラムの２つの行事（朝の会 と 授業） はじめる → 第3回 時間をあやつる命令を使おう（delay） はじめる → 第4回 データをあずかる「魔法の箱」（変数） はじめる → 第5回 箱の中身を入れ替えよう（代入） はじめる → 第6回 コンピューターに計算させよう（演算子） はじめる → 第7回 終わらない授業「loop」のパワー はじめる → 第8回 キーボードの入力を受け取ろう はじめる → 第9回 もし〜なら（条件分岐 if） はじめる → 第10回 そうでなければ（ if - else ） はじめる → 第11回 決まった回数だけ繰り返そう（ for ） はじめる → 第12回 総合演習「入力で切りかわる「クイズ・ボット」を作ろう」 はじめる →
