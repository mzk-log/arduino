基礎編 第14回：LEDの点滅を画面でも見よう（シリアルモニタ／プロッタ）

1. テーマ

第13回と同じオンボードLEDの点滅に加えて、点灯・消灯の状態をシリアルモニタとシリアルプロッタにも送り、光と画面の両方で動きを確認します。

2. 今回覚えること

💡 はじめに読むミニ知識：

ここに出てくる言葉は、このあとの説明やこれからのレッスンで何度も登場します。だから、今は「へぇ、そんな名前の言葉があるんだな」と眺めるだけで、完全に覚えられなくても全然大丈夫です！安心して進めてくださいね。

  • 状態の数値化： 点灯を 1、消灯を 0 のように数字で表すこと。
  • Serial.println（シリアル・プリントエルエヌ）： 値を1行ずつパソコンへ送る関数（これまでもメッセージ表示で使いました）。
  • シリアルプロッタ： 送られてきた数字をグラフ（波形）で見せる画面。

3. 事前解説

第13回では、LEDの光そのものを見ました。今回は、その同じ切り替えを数字にしてパソコンへ送ります。

シリアルモニタでは 1 と 0 が交互に出ます。シリアルプロッタでは、その数字が山と谷のような波形になります。「実物の光」と「画面の数字／波」が同じリズムで動くことを確かめましょう。

まずモニタで文字を確認し、次にプロッタで波形を見る順がおすすめです。モニタとプロッタは同時に開けないことが多いので、片方を閉じてからもう片方を開いてください。

4. コード（プログラム）

初めての方・途中から始める方へ：インデントとカッコのルールを確認

下のアルファベットと記号を、パソコンの画面にそのまま打ち込んでみましょう。

void setup() {
  pinMode(LED_BUILTIN, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  digitalWrite(LED_BUILTIN, HIGH);
  Serial.println(1);
  delay(1000);
  digitalWrite(LED_BUILTIN, LOW);
  Serial.println(0);
  delay(1000);
}

5. 動かし方（手順）

入力したプログラムを、実際に動かしてみましょう！

手順 1 / 準備
Arduinoをパソコンにつなぐ

USBケーブルを使って、Arduinoとパソコンをしっかりとつなぎます。

手順 2 / チェック
チェックボタン（検証）を押す

画面の左上にある「レ点（✓）のボタン」をクリックします。「コンパイル完了」を確認します。

手順 3 / 書き込み
矢印ボタン（書き込み）を押す

「右矢印（➔）のボタン」で書き込み、「書き込みが完了しました」を待ちます。

手順 4 / モニタ
シリアルモニタを開く

右上のシリアルモニタ（または ツール → シリアルモニタ）を開き、通信速度が 9600 baud になっているか確認します。本体LEDの点滅とあわせて、1 と 0 が交互に出るか見ます。

手順 5 / プロッタ
シリアルプロッタを開く

シリアルモニタを閉じ、右上のシリアルプロッタ（または ツール → シリアルプロッタ）を開きます。同じリズムで上下する波形が出るか確認します。

🔍 確認しよう：

LEDの点滅と同じタイミングで、モニタに 1/0 が出て、プロッタでも波がくり返されたら大成功です！

6. コード解説

  • pinMode(LED_BUILTIN, OUTPUT);
    第13回と同じく、オンボードLED用ピンを出力にします。
  • Serial.begin(9600);
    パソコンと通信する準備です。モニタ／プロッタの速さも 9600 に合わせます。
  • digitalWrite(LED_BUILTIN, HIGH);
    LEDを点灯側にします。
  • Serial.println(1);
    点灯を表す数字 1 を1行送ります。プロッタはこの数字をグラフにします。
  • delay(1000);
    1秒待ちます。
  • digitalWrite(LED_BUILTIN, LOW);
    LEDを消灯側にします。
  • Serial.println(0);
    消灯を表す数字 0 を1行送ります。
  • void loop()
    点灯と消灯のセットを、授業としてくり返します。

7. 練習問題

Serial.println(1); と Serial.println(0); の数字を入れ替えて（点灯時に 0、消灯時に 1）、プロッタの波形がどう変わるか観察してみましょう。終わったら元に戻すのも大切です。

8. うまく動かないとき（エラー）のアクション

もしうまく動かない場合は、あわてずに以下の 3つのポイント を手元と見比べて、直してみましょう。

1. 通信速度は 9600 ですか？
モニタ／プロッタの右下付近で 9600 baud になっているか確認してください。
2. モニタとプロッタを同時に開いていませんか？
片方を閉じてから、もう片方を開き直してください。
3. 送っているのは数字だけですか？
プロッタは、余分な文字が多いと波形が見づらくなります。まずはこのレッスンのコードのまま試してください。
