基礎編 第12回：総合演習「入力で切りかわる「クイズ・ボット」を作ろう」

1. テーマ

これまでの学習（表示・変数・入力・if/else・くり返し）を組み合わせて、番号で答えるミニクイズを作り、正解・不正解と得点を画面に出す総合演習です。

2. 今回覚えること

💡 はじめに読むミニ知識：

ここに出てくる言葉は、このあとの説明やこれからのレッスンで何度も登場します。だから、今は「へぇ、そんな名前の言葉があるんだな」と眺めるだけで、完全に覚えられなくても全然大丈夫です！安心して進めてくださいね。

  • 総合演習： これまでに学んだ部品を組み合わせて、ひとつのプログラムを完成させること。
  • 得点（score）： 正解した回数を覚えておく変数。
  • クイズ・ボット： 問題を出し、入力を受け取り、正誤を判定して返事をするプログラム。

3. 事前解説

いよいよ最終回です。シリアルモニタ上で動くクイズ・ボットを作ります。

流れは次のとおりです。

  • 朝の会で問題を出します。
  • 授業で、キーボードから番号（1や2）を受け取ります。
  • if / else で正解か不正解かを判定します。
  • 正解なら得点を増やします。

部品（LEDやモーター）は使いません。文字のやり取りだけで、立派なクイズプログラムになります！

4. コード（プログラム）

初めての方・途中から始める方へ：インデントとカッコのルールを確認

下のアルファベットと記号を、パソコンの画面にそのまま打ち込んでみましょう。

int score = 0;
int question = 1;

void setup() {
  Serial.begin(9600);
  Serial.println("クイズ・ボット起動！");
  Serial.println("Q1: 太陽がのぼる方向は？");
  Serial.println("1: 東");
  Serial.println("2: 西");
}

void loop() {
  if (Serial.available() > 0) {
    char c = Serial.read();

    if (c == '1' || c == '2') {
      if (question == 1) {
        if (c == '1') {
          Serial.println("正解！");
          score = score + 1;
        } else {
          Serial.println("ざんねん！正解は 1: 東");
        }
        question = 2;
        Serial.println("Q2: 水の化学式は？");
        Serial.println("1: CO2");
        Serial.println("2: H2O");
      } else if (question == 2) {
        if (c == '2') {
          Serial.println("正解！");
          score = score + 1;
        } else {
          Serial.println("ざんねん！正解は 2: H2O");
        }
        Serial.print("得点: ");
        Serial.println(score);
        Serial.println("おつかれさま！HOMEへ戻って復習しよう");
        question = 3;
      }
    }
  }
}

5. 動かし方（手順）

入力したプログラムを、実際に動かしてみましょう！

手順 1 / 準備
Arduinoをパソコンにつなぐ

USBケーブルを使って、Arduinoとパソコンをしっかりとつなぎます。

手順 2 / チェック
チェックボタン（検証）を押す

画面の左上にある「レ点（✓）のボタン」をクリックします。これは、プログラムに文字の間違いがないかをパソコンが自動で調べるボタンです。画面の下の方に「コンパイル完了」と出れば、間違いがないという証拠です！

手順 3 / 書き込み
矢印ボタン（書き込み）を押す

間違いがないことを確認したら、隣にある「右矢印（➔）のボタン」をクリックします。プログラムがArduinoへ送り込まれます。画面の下の方に「書き込みが完了しました」と出るまで少し待ちます。

手順 4 / 確認
シリアルモニタを開く

画面の右上にある「虫眼鏡（🔍）のボタン」をクリックして、シリアルモニタという新しい画面を開きます。

🔍 確認しよう：

起動後に問題が出て、「1」→「2」の順に答えると、正誤メッセージのあと「得点: （0〜2）」が表示されたら大成功です！

6. コード解説

  • int score = 0;
    得点を入れる変数です。最初は0です。
  • int question = 1;
    いま何問目かを覚えておく変数です。
  • void setup()
    クイズ開始と第1問の表示を1回だけ行います。
  • if (Serial.available() > 0)
    入力があるときだけ判定します。
  • char c = Serial.read();
    入力された文字（番号）を読みます。
  • if (c == '1' || c == '2')
    1か2のときだけ答えとして扱います。|| は「または」。
  • if (question == 1) ... else if (question == 2)
    いまの問題番号に応じて判定を切り替えます。
  • score = score + 1;
    正解したら得点を1増やします。
  • Serial.print / Serial.println
    メッセージと得点を表示します。
  • ; と { }
    区切りと範囲です。入れ子（中に中）になっているので特に注意します。

7. 練習問題

Q1の正解判定を入れ替えて（わざと違う番号を正解にして）、動きのちがいを観察してみましょう。終わったら元に戻すのも大切です。

8. うまく動かないとき（エラー）のアクション

もし画面にエラー（赤い文字など）が出て動かない場合は、あわてずに以下の 3つのポイント を手元と見比べて、直してみましょう。

1. なみカッコの入れ子は合っていますか？
if が多いので、{ と } の数を数えて確認してください。
2. 文字の番号は '1' ですか？
数字の1（int）ではなく、文字の '1' として比べています。
3. シリアルモニタで1文字ずつ送れていますか？
まずは 1 だけ送って反応を見てみましょう。
