基礎編 第4回：データをあずかる「魔法の箱」（変数）

Arduino基板の準備やソフトのインストールがまだの人へ：先に はじめる前の準備 を見てください。

1. テーマ

数字などのデータを、名前のついた箱（変数）に入れておき、あとから取り出して画面に表示する方法を学びます。

2. 今回覚えること

💡 はじめに読むミニ知識：

ここに出てくる言葉は、このあとの説明やこれからのレッスンで何度も登場します。だから、今は「へぇ、そんな名前の言葉があるんだな」と眺めるだけで、完全に覚えられなくても全然大丈夫です！安心して進めてくださいね。

  • 変数（へんすう）： データを入れておける、名前付きの箱。
  • int（イント）： 整数（小数点のない数字）を入れるための型。
  • 型（かた）： 箱にどんな種類のデータを入れるかを決める印。
  • 初期化（しょきか）： 箱を作ったときに、最初の値を入れること。

3. 事前解説

コンピューターは、数字や文字などのデータをあつかいます。データをしまっておく場所のことを、プログラミングでは変数（へんすう）と呼びます。

変数は「名前のついた魔法の箱」です。箱を作るときは、どんなデータを入れる箱か（型）と箱の名前を書きます。整数なら int を使います。

たとえば int score = 10; は、「score という名前の整数の箱を作り、最初に 10 を入れる」という意味です。箱の中身は Serial.println(score); で画面に出せます。

4. コード（プログラム）

初めての方・途中から始める方へ：インデントとカッコのルールを確認

下のアルファベットと記号を、パソコンの画面にそのまま打ち込んでみましょう。

void setup() {
  Serial.begin(9600);

  int score = 10;
  Serial.println("点数を箱に入れました");
  Serial.println(score);
}

void loop() {
}

5. 動かし方（手順）

入力したプログラムを、実際に動かしてみましょう！

手順 1 / 準備
Arduinoをパソコンにつなぐ

USBケーブルを使って、Arduinoとパソコンをしっかりとつなぎます。

手順 2 / チェック
チェックボタン（検証）を押す

画面の左上にある「レ点（✓）のボタン」をクリックします。これは、プログラムに文字の間違いがないかをパソコンが自動で調べるボタンです。画面の下の方に「コンパイル完了」と出れば、間違いがないという証拠です！

手順 3 / 書き込み
矢印ボタン（書き込み）を押す

間違いがないことを確認したら、隣にある「右矢印（➔）のボタン」をクリックします。プログラムがArduinoへ送り込まれます。画面の下の方に「書き込みが完了しました」と出るまで少し待ちます。

手順 4 / 確認
シリアルモニタを開く

画面の右上にある「虫眼鏡（🔍）のボタン」をクリックして、シリアルモニタという新しい画面を開きます。

🔍 確認しよう：

「点数を箱に入れました」のあとに「10」と表示されたら大成功です！

6. コード解説

  • void setup() { ... }
    朝の会。今回の処理はここで1回だけ行います。
  • Serial.begin(9600);
    通信の準備です。
  • int score = 10;
    int 型の変数 score を作り、最初の値として 10 を入れます（初期化）。
  • = （イコール）
    ここでは「等しい」ではなく、「右の値を左の箱に入れる」という意味で使います。
  • Serial.println("点数を箱に入れました");
    説明の文字列を表示します。
  • Serial.println(score);
    変数 score の中身（10）を表示します。文字列ではないので " はつけません。
  • ;
    処理の区切りです。
  • void loop() { }
    授業は今回空っぽです。
  • { と }
    関数の範囲です。

7. 練習問題

int score = 10; の数字を 25 に変えて、画面に「25」が出るか確かめてみましょう。

8. うまく動かないとき（エラー）のアクション

もし画面にエラー（赤い文字など）が出て動かない場合は、あわてずに以下の 3つのポイント を手元と見比べて、直してみましょう。

1. 変数の型と名前は半角ですか？
int や score が全角になっていないか確認してください。
2. セミコロンを忘れていませんか？
int score = 10 の最後に ; があるか確認してください。
3. Serial.println(score); に余計な " をつけていませんか？
変数の中身を出すときは "score" ではなく score です。
