基礎編 第6回：コンピューターに計算させよう（演算子）

Arduino基板の準備やソフトのインストールがまだの人へ：先に はじめる前の準備 を見てください。

1. テーマ

変数に入れた数字どうしを足し算・引き算・かけ算・わり算し、その結果を画面に表示する方法を学びます。

2. 今回覚えること

💡 はじめに読むミニ知識：

ここに出てくる言葉は、このあとの説明やこれからのレッスンで何度も登場します。だから、今は「へぇ、そんな名前の言葉があるんだな」と眺めるだけで、完全に覚えられなくても全然大丈夫です！安心して進めてくださいね。

  • 演算子（えんざんし）： 計算に使う記号。+ - * / など。
  • +： 足し算。
  • -： 引き算。
  • *： かけ算。
  • /： わり算（整数どうしでは、あまりは切り捨てられます）。

3. 事前解説

変数に入れた数字は、そのまま表示するだけでなく、計算にも使えます。計算に使う記号を演算子と呼びます。

たとえば a + b は「a と b を足す」という意味です。計算結果も、変数に代入したり、そのまま画面に出したりできます。

注意：int（整数）どうしのわり算では、小数点以下は切り捨てられます。例：7 / 2 の結果は 3.5 ではなく 3 になります。

4. コード（プログラム）

初めての方・途中から始める方へ：インデントとカッコのルールを確認

下のアルファベットと記号を、パソコンの画面にそのまま打ち込んでみましょう。

void setup() {
  Serial.begin(9600);

  int a = 7;
  int b = 2;

  Serial.println("足し算");
  Serial.println(a + b);

  Serial.println("引き算");
  Serial.println(a - b);

  Serial.println("かけ算");
  Serial.println(a * b);

  Serial.println("わり算");
  Serial.println(a / b);
}

void loop() {
}

5. 動かし方（手順）

入力したプログラムを、実際に動かしてみましょう！

手順 1 / 準備
Arduinoをパソコンにつなぐ

USBケーブルを使って、Arduinoとパソコンをしっかりとつなぎます。

手順 2 / チェック
チェックボタン（検証）を押す

画面の左上にある「レ点（✓）のボタン」をクリックします。これは、プログラムに文字の間違いがないかをパソコンが自動で調べるボタンです。画面の下の方に「コンパイル完了」と出れば、間違いがないという証拠です！

手順 3 / 書き込み
矢印ボタン（書き込み）を押す

間違いがないことを確認したら、隣にある「右矢印（➔）のボタン」をクリックします。プログラムがArduinoへ送り込まれます。画面の下の方に「書き込みが完了しました」と出るまで少し待ちます。

手順 4 / 確認
シリアルモニタを開く

画面の右上にある「虫眼鏡（🔍）のボタン」をクリックして、シリアルモニタという新しい画面を開きます。

🔍 確認しよう：

足し算9、引き算5、かけ算14、わり算3 と表示されたら大成功です！

6. コード解説

  • int a = 7; / int b = 2;
    計算用の変数を2つ作ります。
  • a + b
    足し算の結果（9）を作ります。
  • a - b
    引き算の結果（5）です。
  • a * b
    かけ算の結果（14）です。プログラムでは × の代わりに * を使います。
  • a / b
    わり算の結果です。整数どうしなので 3 になります。
  • Serial.println(...);
    計算結果や説明文を表示します。
  • void setup() / void loop() と ; { }
    朝の会・授業、区切り、範囲です。

7. 練習問題

int a = 7; を int a = 10; に変えて、それぞれの計算結果がどう変わるか確かめてみましょう。

8. うまく動かないとき（エラー）のアクション

もし画面にエラー（赤い文字など）が出て動かない場合は、あわてずに以下の 3つのポイント を手元と見比べて、直してみましょう。

1. かけ算は * ですか？
× や x ではなく、半角の * を使います。
2. セミコロンを忘れていませんか？
各行の終わりに ; があるか確認してください。
3. 変数名は半角英数字ですか？
a や b が全角になっていないか確認してください。
