基礎編 第11回：決まった回数だけ繰り返そう（ for ）

Arduino基板の準備やソフトのインストールがまだの人へ：先に はじめる前の準備 を見てください。

1. テーマ

始める値・終わる条件・1回ごとの変化をまとめて書き、決めた回数だけ同じ処理をくり返す方法を学びます。

2. 今回覚えること

💡 はじめに読むミニ知識：

ここに出てくる言葉は、このあとの説明やこれからのレッスンで何度も登場します。だから、今は「へぇ、そんな名前の言葉があるんだな」と眺めるだけで、完全に覚えられなくても全然大丈夫です！安心して進めてくださいね。

  • for： 決まった回数だけくり返すための構文。
  • カウンタ変数： いま何回目かを数えるための変数（よく i を使います）。
  • i++： i = i + 1 と同じ意味（1増やす）。

3. 事前解説

loop は「終わりなくくり返す授業」でした。それに対して for は、「何回くり返すか」を決めてから回す道具です。 for (int i = 1; i <= 5; i++) は次の3つをまとめて書きます。
  • はじめ：i = 1
  • 続く条件：i <= 5（5以下なら続ける）
  • 1回ごとの変化：i++（1増やす）

朝の会のあいだに「1から5まで番号を読む」イメージです。

4. コード（プログラム）

初めての方・途中から始める方へ：インデントとカッコのルールを確認

下のアルファベットと記号を、パソコンの画面にそのまま打ち込んでみましょう。

void setup() {
  Serial.begin(9600);
  Serial.println("1から5まで数えます");

  for (int i = 1; i <= 5; i++) {
    Serial.println(i);
    delay(500);
  }

  Serial.println("終わり！");
}

void loop() {
}

5. 動かし方（手順）

入力したプログラムを、実際に動かしてみましょう！

手順 1 / 準備
Arduinoをパソコンにつなぐ

USBケーブルを使って、Arduinoとパソコンをしっかりとつなぎます。

手順 2 / チェック
チェックボタン（検証）を押す

画面の左上にある「レ点（✓）のボタン」をクリックします。これは、プログラムに文字の間違いがないかをパソコンが自動で調べるボタンです。画面の下の方に「コンパイル完了」と出れば、間違いがないという証拠です！

手順 3 / 書き込み
矢印ボタン（書き込み）を押す

間違いがないことを確認したら、隣にある「右矢印（➔）のボタン」をクリックします。プログラムがArduinoへ送り込まれます。画面の下の方に「書き込みが完了しました」と出るまで少し待ちます。

手順 4 / 確認
シリアルモニタを開く

画面の右上にある「虫眼鏡（🔍）のボタン」をクリックして、シリアルモニタという新しい画面を開きます。

🔍 確認しよう：

「1から5まで数えます」のあと、1〜5が表示され、最後に「終わり！」と出たら大成功です！

6. コード解説

  • for (int i = 1; i <= 5; i++)
    1から始めて、5以下のあいだくり返し、毎回1増やします。
  • int i = 1
    カウンタ変数 i を1で始めます。
  • i <= 5
    i が5以下ならくり返しを続けます。<= は「以下」。
  • i++
    i を1増やします。
  • Serial.println(i);
    いまの番号を表示します。
  • delay(500);
    0.5秒待って見やすくします。
  • void setup() / void loop() と ; { }
    朝の会・授業、区切り、範囲です。

7. 練習問題

i <= 5 を i <= 10 に変えて、1から10まで表示されるか確かめてみましょう。

8. うまく動かないとき（エラー）のアクション

もし画面にエラー（赤い文字など）が出て動かない場合は、あわてずに以下の 3つのポイント を手元と見比べて、直してみましょう。

1. for の丸カッコ内はセミコロンで区切られていますか？
for (はじめ; 条件; 変化) の形です。
2. <= は半角ですか？
全角の ≤ などは使えません。
3. なみカッコはそろっていますか？
for の { } を確認してください。
