リファレンスで調べよう

レッスンに出てこない言葉や関数に出会ったら、自分で調べる力が役立ちます。ここでは調べ方の手順を練習します。

💡 はじめに読むミニ知識：

リファレンスは「答え丸写し用の教科書」ではなく、辞書です。説明を読んで、自分のプログラムで小さく試すのがコツです。

2. 調べる4ステップ

手順 1
キーワードを決める

調べたい言葉を決めます。例：delay、digitalWrite、int、if

手順 2
索引や検索から探す

リファレンスのページで、索引やページ内検索を使ってキーワードを見つけます。アルファベット順の一覧があることも多いです。

手順 3
説明と例を読む

次の点に注目します。

  • 何をする関数／型か（役割）
  • まるカッコに何をわたすか（材料・引数）
  • 何か値を返すか（戻り値）
  • 使用例のコード
手順 4
自分のコードで小さく試す

いきなり長いプログラムを変えず、レッスンの短いコードで1か所だけ試します。動いたらメモを残しましょう。

3. 練習（やってみよう）

次のどれか1つを選んで、リファレンスで調べてみてください。

  • delay … 待つ時間の単位は何？（ヒント：ミリ秒）
  • pinMode … わたす材料はいくつ？
  • Serial.println … print とのちがいは？

調べたら、ノートやメモ帳に「役割」「わたすもの」「注意点」を自分の言葉で3行書いてみましょう。

4. うまく見つからないとき

  • つづりを確認する（大文字・小文字）
  • 似た言葉で探す（例：表示 → print / println）
  • レッスン本文の「今回覚えること」に戻る
  • それでも不明なら、保護者や先生と一緒にキーワードを決める
