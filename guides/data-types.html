データ型を知ろう

変数という「箱」には、入れるデータの種類に合わせた型（かた）があります。基礎編でよく使う型を整理します。

💡 はじめに読むミニ知識：

全部を暗記する必要はありません。「数字用」「1文字用」「結果を返さない」くらいの違いが分かれば大丈夫です。くわしく調べたくなったら、リファレンスで調べようへ進みましょう。

1. 型ってなに？

箱（変数）を作るとき、コンピューターに「この箱にはどんな種類のデータを入れる予定？」と伝えます。その種類の名前がデータ型です。

例：

int score = 0;
char c = '1';

int は「整数を入れる箱」、char は「1文字を入れる箱」という意味です。

2. 基礎編でよく使う型

int（イント）

  • 何を入れられる？ 整数（小数点のない数）。例：0、12、-3
  • どんなときに使う？ 得点、回数、待ち時間の計算など、数を数えたり比べたりするとき
  • 基礎編の例： 第4回以降の変数、score など
補足（Arduino Uno / Nano など）：

int はだいたい −32768〜32767 の整数を入れられます。もっと大きな数が必要なら、あとで long などを学びます。

char（チャー／キャラ）

  • 何を入れられる？ 1文字。例：'A'、'1'、'\n'（改行）
  • どんなときに使う？ キーボードから1文字受け取ったときなど
  • 基礎編の例： 第8回以降の Serial.read()、char c = Serial.read();

文字は シングルクォーテーション（ ' ）で囲みます。文章（文字列）の " とはちがいます。

char c = '1';     // 文字の1
int n = 1;         // 数字の1

見た目は似ていても、'1' と 1 は別物です。クイズ・ボットでは文字の番号と比べています。

void（ボイド）

  • 意味： 「空っぽ（結果を返さない）」
  • どんなときに使う？ setup や loop のように、仕事をするだけで値を返さない関数の頭につける

void は変数の箱というより、「この関数は答えを返さないよ」という印です。

3. くらべてみよう

  • int … 数を入れる（計算・カウント向き）
  • char … 1文字を入れる（キー入力向き）
  • void … 関数が結果を返さないときの印

4. これから出会うかもしれない型（参考）

今は名前だけ知っておけば十分です。

  • bool … 真（true）か偽（false）だけ
  • float … 小数のある数
  • long … int より大きな整数
  • String … 複数の文字をつなげた文章（扱い方が少し特別）

5. 迷ったときのコツ

  • 数えたい・計算したい → まず int
  • キーボードの1文字 → char
  • setup / loop → 頭は void
